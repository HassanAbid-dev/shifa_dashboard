# Tailwind Color Scheme

| Element                      | Tailwind Class               |
| ---------------------------- | ---------------------------- |
| Page background              | bg-slate-950                 |
| Sidebar background           | bg-slate-900                 |
| Cards/panels                 | bg-slate-800                 |
| Borders                      | border-slate-700             |
| Primary text                 | text-slate-100               |
| Muted text                   | text-slate-400               |
| Active nav highlight         | bg-blue-900 text-blue-400    |
| Accent blue (buttons, links) | text-blue-400 or bg-blue-600 |
| High alert (red)             | text-red-400                 |
| Medium alert (orange)        | text-amber-400               |
| Low alert (yellow/green)     | text-green-400               |
| Online dot (green)           | bg-green-500                 |
| Offline dot (red)            | bg-red-500                   |
